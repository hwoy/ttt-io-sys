
# ttt-io-sys

ttt-sys is just a low level io ffi-library for Tic-Tac-Toe game in Rust.

## How to build

```sh

git clone --recurse-submodules https://github.com/hwoy/ttt-io-sys.git
cd ttt-io-sys
cargo build

```
